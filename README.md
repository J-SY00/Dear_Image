## 프로젝트 설명
- 색깔 바꾸기
    세부영역 선택하여 바꿀 수 있게
- 배경 지우기 (모자이크)
- 오브젝트 추가
- 오브젝트 삭제
- 오브젝트 크기 조정
- 레퍼런스 올렸을 때 텍스트 비슷하게 이미지 만들어주기
    (INPUT) 레퍼런스이미지(글꼴, 색깔, 크기, 위치), 텍스트내용, 수정할 이미지
    → (OUTPUT) 이미지
